# remnote-epub — Premium EPUB 3.3 from RemNote-style book analysis

> Convert RemNote markdown analysis files — with Essence Lens, flashcards, and bilingual glossary — into beautifully styled EPUB 3.3. Universal for any book or topic analysis.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
remnote-epub v2.0 is a complete rewrite that converts RemNote-style markdown analysis files into premium EPUB 3.3 ebooks. It handles the full range of RemNote formatting: `( __italic__ )`, **bold**, TAB hierarchy, essence boxes, flashcard blocks, misconceptions, cross-domain bridges, and bilingual glossary entries. Each block type gets its own visual treatment. Designed for book analysis with Essence Lens format — not for novels or raw markdown (use epub-forge or epub-builder for those).

## Features
- **Essence boxes** — warm cream background (#f8f5f0), amber left border (#C9A96E)
- **Flashcard blocks** — cool blue background (#f0f4f8), blue left border (#4a90d9)
- **Handles** — `( __italic__ )`, bold, TAB hierarchy, glossary entries, misconceptions, cross-domain bridges
- **v2.0** — universal rewrite, works for any book/topic analysis (not Abhidhamma-only)
- **EPUB 3.3** — W3C compliant output
